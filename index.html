📖 Panduan Setup & Deploy
Cara setup database dan deploy ke GitHub Pages
💾 Pilihan Penyimpanan Data
RECOMMENDED
Supabase (PostgreSQL)

Database PostgreSQL gratis, real-time, mudah setup. Cocok untuk GitHub Pages.

1. Buat akun di supabase.com
2. Buat project baru
3. Copy URL & anon key
4. Isi di SUPABASE_CONFIG di bawah
Firebase Firestore (Google)

NoSQL real-time database dari Google. Perlu setup lebih, tapi lebih dikenal.

💻 LocalStorage (Offline Mode)

Data tersimpan di browser lokal. Tidak perlu setup apapun, tapi data hilang jika clear browser atau ganti perangkat.

⚠️ Saat ini berjalan di LocalStorage. Ganti ke Supabase untuk data permanen & multi-device.
🚀 Deploy ke GitHub Pages
GitHub Pages adalah hosting gratis dari GitHub untuk file HTML/CSS/JS statis.
1
Buat Repository GitHub
Buka github.com → New Repository → Nama: hidroponik
2
Upload File
Drag & drop index.html ke repository. Pastikan nama filenya tepat.
3
Aktifkan GitHub Pages
Settings → Pages → Source: Deploy from branch → Branch: main → Save
4
Website Live!
URL: username.github.io/hidroponik
🔧 Setup Supabase (Database Gratis)
Langkah Setup Supabase

1. Buka supabase.com → Sign Up → Buat Project Baru

2. Di dashboard, buka SQL Editor dan jalankan query ini:

-- Buat tabel
CREATE TABLE batches (
  id SERIAL PRIMARY KEY,
  batch_id INTEGER,
  date_semai DATE,
  qty INTEGER,
  note TEXT,
  date_pindah DATE,
  date_panen DATE,
  qty_rakit INTEGER,
  est_kg DECIMAL,
  status TEXT DEFAULT 'semai',
  actual_kg DECIMAL,
  harvest_note TEXT,
  actual_harvest_date DATE,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE expenses (
  id SERIAL PRIMARY KEY,
  date DATE, cat TEXT, desc TEXT,
  amount DECIMAL, rencana_id INTEGER,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE sales (
  id SERIAL PRIMARY KEY,
  invoice TEXT, date DATE, buyer TEXT,
  qty DECIMAL, price DECIMAL, total DECIMAL,
  pay TEXT, due DATE, note TEXT,
  status TEXT, paid_date DATE,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE rencana (
  id SERIAL PRIMARY KEY,
  month TEXT, cat TEXT, desc TEXT,
  amount DECIMAL,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

CREATE TABLE settings (
  key TEXT PRIMARY KEY,
  value JSONB,
  updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- Enable Row Level Security (public access)
ALTER TABLE batches ENABLE ROW LEVEL SECURITY;
ALTER TABLE expenses ENABLE ROW LEVEL SECURITY;
ALTER TABLE sales ENABLE ROW LEVEL SECURITY;
ALTER TABLE rencana ENABLE ROW LEVEL SECURITY;
ALTER TABLE settings ENABLE ROW LEVEL SECURITY;

-- Allow all (ubah setelah auth disetup)
CREATE POLICY "Public" ON batches FOR ALL USING (true);
CREATE POLICY "Public" ON expenses FOR ALL USING (true);
CREATE POLICY "Public" ON sales FOR ALL USING (true);
CREATE POLICY "Public" ON rencana FOR ALL USING (true);
CREATE POLICY "Public" ON settings FOR ALL USING (true);

3. Buka Settings → API → Copy URL & anon key

4. Paste ke SUPABASE_CONFIG di file index.html

Template Konfigurasi
const SUPABASE_CONFIG = {
  url: "https://xxx.supabase.co",
  anon_key: "eyJhbGc..."
};
Gratis Selamanya: Supabase tier gratis cukup untuk usaha kecil — 500MB database, 2GB transfer/bulan.
